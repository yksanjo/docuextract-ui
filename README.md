# DocuExtract UI

Web Dashboard for DocuExtract Gateway.

## Features

- Provider health status monitoring
- Document upload and extraction
- Real-time usage statistics
- Extracted text preview

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:3001 to view the dashboard.

## Configuration

Set the gateway URL in the API calls (default: http://localhost:3000)
